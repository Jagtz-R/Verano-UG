# Verano-UG
El presente repositorio contiene la base de la aplicación para el projecto de "Control Remoto para un Robot Planar Serial de 2 GDL". 
Este proyecto esta desarrollado para la plataforma Esp32 y comprende lo siguiente: Enlace WiFi, Comunicación con un servidor MQTT, rutinas de atención a comandos enviados por MQTT, rutina de comunicación serial con el controlador del robot y publicación de respuestas al servidor MQTT.
